hypochondriapp
About

Feeling sick? Enter your symptoms. Get back the scariest sounding disease possible.

Hypochondriapp is a spoof on the modern phenomenon of web-induced hypochondria. It was made by Emily Xie for the 2017 Stupid Shit No One Needs and Terrible Ideas Hackathon

Hypochondriapp was built using Python, Javascript, JQuery, and Flexbox. It scrapes Wikipedia's rare diseases section, and simply returns one of the diseases with the top 3 longest names that matches any one of your entered symptoms.

See anything broken / funky? Feel free to contact me via Twitter or file an issue on Github. This app was made in a rush and probably (definitely) contains many errors. Also, the code is terrible.

The app is a remake of an old project I did back in 2013, with collaborators Greg Eng and David Bella.
